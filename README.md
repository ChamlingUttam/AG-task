 #  Frontend Intern Task - Country Explorer 



## Project OverView

This project is two web page applictaion build using React.js.This project demonstrate smooth animation,Api integration and deployment. This projects allows user to explore details about specfice information of all country over the world.

## Objectives

- Responsive designe
- Api integration 
- Deployment
- Animation and interaction


## live demo 

- this project was deplyed using vercel

- Link => https://ag-task.vercel.app/


## Tech Used 
- React.js 
- Tailwind Css
- Axios 
- Zustand
- Frmaer motion 

## Api Used

-  REST Countries API  
  https://restcountries.com/v3.1/all?fields=name,flags,population,region,cca3

## 🚀 Installation & Setup

```bash
# Clone the repo
git clone https://github.com/ChamlingUttam/AG-task.git

# Go to project folder
cd AG-task

# Install dependencies
npm install

# Run dev server
npm run dev


## Design Decision
- Used tailwind css for styling
- used axios to handle api calling
- used zustand for lightweight global state managment 
- used framer-motion for smooth animation 

## Challange faced 

- I was unware about the requesting all data of countries without specifing fileds thats how i came up with the idea to integrate for specific infromation.
